### Source code for the two algorithms that produce unit-distance graphs and for procedures that analyze their properties.

* **unit-distance-greedy_search_v1.py** 
  * A greedy algorithm that explores the rational plane and generates dense and highly symmetrical unit-distance graphs.

* **unit-distance-heuristic_search_v1.py** 
  * A heuristic algorithm that explores the rational plane and generates grid-based dense unit-distance graphs.

* **statistics.py** 
  * A python script that analyzes the produced graph nodes calculates the unit-distance pairs among them, their density and plots the results.

* **Validation.java** 
  * validates the produced results
  * calculates statistics on the graph elements
  * stores nodes in file of the form <id, x_num, x_den, y_num, y_den >
  * stores unit-distance pairs in edge list format <node1, node2>
