### Experimental results for the greedy search algorithm.

* **greedy-search-results_509K.csv** 
  * The produced 509,445 points in the form <x_num, x_den, y_num, y_den, x_float, y_float>.

* **greedy-search-metrics.csv** 
  * Statistics of the generated graph.

* **greedy-search-logfile.txt** 
  * The log file generated during execution of the script.

* **greedy-graph-nodes.csv** 
  * The 509,445 graph nodes stored in the form <id, x_num, x_den, y_num, y_den >

* **greedy-graph-edges.csv** 
  * The 4,919,523 graph edges stored in edge list format <node1, node2>
