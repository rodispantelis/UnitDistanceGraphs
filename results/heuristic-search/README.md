### Experimental results for the heuristic search algorithm.

* **heuristic-search-results_22K.csv** 
  * The produced 22,030 points stored in the form <x_num, x_den, y_num, y_den, x_float, y_float>.

* **heuristic-search-metrics.csv** 
  * Statistics of the generated graph.

* **heuristic-graph-nodes.csv** 
  * The 22,030 graph nodes stored in the form <id, x_num, x_den, y_num, y_den>

* **heuristic-graph-edges.csv** 
  * The 123,101 graph edges stored in edge list format <node1, node2>
